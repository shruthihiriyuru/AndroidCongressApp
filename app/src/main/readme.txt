ReadeMe File for Congress App

This folder contains the complete implementation of the Congress Application for Android platform demonstrating various functionalities.

The files contained here are:
AndroidManifest.xml

Java Files:

MainActivity.java:
- Routine that controls the entire process.
- Creates the initial screen, Sets controls for Navigation Drawer
- Initializes Navigation Drawer and its layout
- Inflates Legislator fragment as the main fragment

AboutMeActivity.java - Display student details

Fragment Files:
- LegislatorFragment, BillFragment, CommitteeFragment and FavoriteFragment for handling the display of each sections.


Display Activities:

- Displays activities for showing the detais of each section - Legislators, Bills, Committees and Favorites.

AboutMeActivity:
- Displays details of developer.
